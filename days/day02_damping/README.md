# Day 2: Damping

## Files

- `model.py`: placeholder for the damping solver
- `notes.md`: Day 2 notes

## Simulation

This module solves the damped oscillator from:

```text
m, b, k, x(0), v(0)
```

It classifies the motion as undamped, underdamped, critical, or overdamped and
returns sampled points for the browser.
