# Coupled Oscillators Without Damping

Equations:

```text
m x1'' = -k x1 - kc(x1 - x2)
m x2'' = -k x2 - kc(x2 - x1)
```

Normal coordinates:

```text
q+ = (x1 + x2) / 2
q- = (x1 - x2) / 2
```

Normal modes:

```text
omega_in = sqrt(k / m)
omega_out = sqrt((k + 2kc) / m)
```

Any motion is a mixture of the two modes.
